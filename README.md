# Git Basics Tutorial | آموزش مقدماتی Git

## 📖 Overview

This repository contains two educational text files that introduce the essential concepts and commands of **Git**. The content is provided in both **English** and **Persian (Farsi)** to make learning easier for a wider audience.

این مخزن شامل دو فایل آموزشی متنی درباره مفاهیم پایه و دستورات ضروری **Git** است. مطالب به دو زبان **انگلیسی** و **فارسی** تهیه شده‌اند تا یادگیری برای کاربران بیشتری امکان‌پذیر باشد.

## 📂 Repository Contents

* `Git-English.txt` – Git tutorial and essential commands in English.
* `Git-Persian.txt` – آموزش Git و دستورات ضروری به زبان فارسی.

## 📚 Topics Covered

* What is Git?
* Installing Git
* Initializing a repository
* Git workflow
* Essential Git commands
* Branching basics
* Committing changes
* Working with remote repositories
* GitHub basics
* Best practices

## 🎯 Purpose

This repository is intended for beginners who want to learn the fundamentals of Git and GitHub through simple, easy-to-read text documents.

هدف این مخزن، آموزش مفاهیم پایه Git و GitHub به کاربران مبتدی با استفاده از فایل‌های متنی ساده و قابل فهم است.
